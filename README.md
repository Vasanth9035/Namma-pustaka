# Namma-Pustaka 📚

## Smart Digital Library Android Application

Namma-Pustaka is a modern Android-based digital library management application developed using Kotlin and Android Studio. The project aims to simplify traditional library systems through smart digital solutions such as QR-based book tracking, catalog management, and reading analytics.

The application is specially designed for schools, colleges, and rural libraries to promote digital literacy and efficient book management.

---

## 🚀 Features

- 📖 Digital Book Catalog
- 🔍 Smart Book Search
- 📱 QR Code Book Tracking
- ⭐ Book Reviews and Ratings
- 📊 Reading Activity Monitoring
- 🏆 Reading Leaderboard
- 🎨 Modern Minimal UI Design
- 📚 Library Management System
- 👨‍🎓 Student-Friendly Interface

---

## 🛠️ Technologies Used

- Kotlin
- Android Studio
- XML
- RecyclerView
- Room Database
- Material Design
- ML Kit QR Scanner
- Git & GitHub

---

## 📌 Problem Statement

Traditional library systems are difficult to manage manually and often lack digital accessibility for students and teachers. Namma-Pustaka solves this problem by providing a smart Android-based library management solution with digital cataloging and QR-based book tracking.

---

## 🎯 Objective

- To digitize traditional library systems
- To encourage reading habits among students
- To simplify book tracking and management
- To provide an easy-to-use mobile library platform

---

## 📱 Application Modules

### 1. Home Module
Displays featured books, categories, and quick access options.

### 2. Book Catalog Module
Allows users to browse and search books easily.

### 3. QR Scanner Module
Helps in tracking books using QR code scanning.

### 4. Reviews & Ratings Module
Users can review and rate books.

### 5. Reading Analytics Module
Tracks reading activities and leaderboard statistics.
